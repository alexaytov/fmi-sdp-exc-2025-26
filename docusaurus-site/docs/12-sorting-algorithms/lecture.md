### 5.1 Merge Sort

<InfoBox title="–ö–æ–Ω—Ü–µ–ø—Ü–∏—è">

–†–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Ä–∞–∑–¥–µ–ª—è–Ω–µ –Ω–∞ –º–∞—Å–∏–≤–∞, —Å–æ—Ä—Ç–∏—Ä–∞–Ω–µ –Ω–∞ –¥–≤–µ—Ç–µ –ø–æ–ª–æ–≤–∏–Ω–∏, —Å–ª–∏–≤–∞–Ω–µ (merge).

**–ö–∞–∫ —Ä–∞–±–æ—Ç–∏ Merge Sort:**

1. **Divide**: –†–∞–∑–¥–µ–ª—è–Ω–µ –Ω–∞ –º–∞—Å–∏–≤–∞ –Ω–∞ –¥–≤–µ –ø–æ–ª–æ–≤–∏–Ω–∏
2. **Conquer**: –†–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Å–æ—Ä—Ç–∏—Ä–∞–Ω–µ –Ω–∞ –≤—Å—è–∫–∞ –ø–æ–ª–æ–≤–∏–Ω–∞
3. **Combine**: –°–ª–∏–≤–∞–Ω–µ –Ω–∞ —Å–æ—Ä—Ç–∏—Ä–∞–Ω–∏—Ç–µ –ø–æ–ª–æ–≤–∏–Ω–∏

<CollapsibleSection title="üìù –ò–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è" icon="üíª">

```cpp
void merge(std::vector<int>& arr, int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;

    // –í—Ä–µ–º–µ–Ω–Ω–∏ –º–∞—Å–∏–≤–∏
    std::vector<int> L(n1), R(n2);

    for (int i = 0; i < n1; i++)
        L[i] = arr[left + i];
    for (int j = 0; j < n2; j++)
        R[j] = arr[mid + 1 + j];

    // –°–ª–∏–≤–∞–Ω–µ –Ω–∞ –≤—Ä–µ–º–µ–Ω–Ω–∏—Ç–µ –º–∞—Å–∏–≤–∏ –æ–±—Ä–∞—Ç–Ω–æ –≤ arr
    int i = 0, j = 0, k = left;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) {
            arr[k] = L[i];
            i++;
        } else {
            arr[k] = R[j];
            j++;
        }
        k++;
    }

    // –ö–æ–ø–∏—Ä–∞–Ω–µ –Ω–∞ –æ—Å—Ç–∞–Ω–∞–ª–∏—Ç–µ –µ–ª–µ–º–µ–Ω—Ç–∏
    while (i < n1) {
        arr[k] = L[i];
        i++;
        k++;
    }
    while (j < n2) {
        arr[k] = R[j];
        j++;
        k++;
    }
}

void mergeSort(std::vector<int>& arr, int left, int right) {
    if (left < right) {
        int mid = left + (right - left) / 2;

        mergeSort(arr, left, mid);
        mergeSort(arr, mid + 1, right);

        merge(arr, left, mid, right);

}
```

**–ö–∞–∫ —Ä–∞–±–æ—Ç–∏ Quicksort:**

1. **–ò–∑–±–æ—Ä –Ω–∞ pivot**: –û–±–∏–∫–Ω–æ–≤–µ–Ω–æ –ø–æ—Å–ª–µ–¥–Ω–∏—è—Ç, –ø—ä—Ä–≤–∏—è—Ç, —Å–ª—É—á–∞–µ–Ω –∏–ª–∏ median-of-three
2. **Partition**: –ü—Ä–µ–Ω–∞—Ä–µ–∂–¥–∞–Ω–µ —Ç–∞–∫–∞ —á–µ –µ–ª–µ–º–µ–Ω—Ç–∏—Ç–µ < pivot —Å–∞ –≤–ª—è–≤–æ, >= pivot —Å–∞ –≤–¥—è—Å–Ω–æ
3. **–†–µ–∫—É—Ä—Å–∏—è**: –°–æ—Ä—Ç–∏—Ä–∞–Ω–µ –Ω–∞ –¥–≤–µ—Ç–µ —á–∞—Å—Ç–∏

```cpp
int partition(std::vector<int>& arr, int low, int high) {
    int pivot = arr[high];  // –∏–∑–±–æ—Ä –Ω–∞ pivot
    int i = low - 1;        // –∏–Ω–¥–µ–∫—Å –Ω–∞ –ø–æ-–º–∞–ª—ä–∫ –µ–ª–µ–º–µ–Ω—Ç

    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            std::swap(arr[i], arr[j]);
        }
    }
    std::swap(arr[i + 1], arr[high]);
    return i + 1;
}

void quickSort(std::vector<int>& arr, int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);

        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
```
